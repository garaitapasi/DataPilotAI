🚀 DataPilot AI

An AI-powered conversational Data Science assistant built with Streamlit, Machine Learning, and LLMs.
DataPilot AI helps users analyze datasets, preprocess data, train ML models, evaluate performance, generate AI insights, and create professional PDF reports — all through a simple chat interface.
